# Tincraft Classic

Tincraft Classic is a Forge 1.12.2 version of the Tincraft series. It brings content from both Tincraft and Tincraft 2 to old versions of the game.

## Features

- Content from Tincraft and Tincraft 2
- New blocks and items
- Block entities
- Forge Support
- Additional improvements and new content

## Why download Tincraft Classic?

Tincraft Classic continues the Tincraft experience on old Minecraft versions with Forge support and ongoing development.

## Important Information

- Forge is required.
- Tincraft Classic is based on the Tincraft series and is developed by the original author.
- Some internal systems may differ from previous Tincraft releases.
- Tincore Can't use Tincore due to it being on modern versions of the game.

## Changes from Tincraft 2

- Forge-only
- Can't use tincore
- Updated for legacy Minecraft versions
- Ongoing development and improvements
